# Tetrix

A capstone project

## Development

- in `/server`, run `pnpm start` or `yarn start` (also try `pnpm ts-node index.ts` if the other two don't work)
- in `/client`, run `pnpm dev` or `yarn dev` (Server takes the PORT env var from console. try `export PORT = 3001`)
